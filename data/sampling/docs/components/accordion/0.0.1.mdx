## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-accordion
```


## Anatomy

Import all parts and piece them together.

```jsx
import * as Accordion from '@radix-ui/react-accordion';

export default () => (
  <Accordion.Root>
    <Accordion.Item>
      <Accordion.Header>
        <Accordion.Trigger />
      </Accordion.Header>
      <Accordion.Content />
    </Accordion.Item>
  </Accordion.Root>
);
```
